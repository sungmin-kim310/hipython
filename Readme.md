## 🚀 프로젝트

### 📊 데이터 분석 및 AI 기반 서비스 개발  
**(삼정 KPMG Future Academy 8기)**

- **프로젝트 개요**
  - 실제 기업 업무 사례를 기반으로 한 데이터 분석 과제 수행
  - AI를 활용해 기존 수작업·판단 중심 업무를 자동화·보조하는 서비스 설계
  - 단순 분석 결과가 아닌 “서비스 형태의 결과물” 도출을 목표로 진행

- **주요 역할**
  - 데이터 분석 관점에서 서비스 기획 참여
  - AI 적용 가능 영역 도출 및 활용 시나리오 설계
  - 데이터 처리 흐름 및 분석 구조 설계
  - 비즈니스 요구사항을 데이터·모델 관점으로 재정의

- **주요 기능**
  - 정형·비정형 데이터 분석
  - 규칙 기반 로직과 AI 판단 로직의 결합
  - 분석 결과 요약 및 인사이트 자동 도출
  - 향후 고도화를 고려한 확장형 구조 설계

- **학습 및 성과**
  - 데이터 분석 → AI 적용 → 서비스 설계까지의 전체 흐름 경험
  - AI를 “정답 생성 도구”가 아닌 **업무 판단을 보조하는 도구**로 활용
  - 컨설팅·기업 실무에서 활용 가능한 데이터 사고방식 체득
