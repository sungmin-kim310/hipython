# 🧾 까먹 (Kka-Meok)
**중소 엔터테인먼트 기업을 위한 영수증 정산 프로세스 AI 보조 시스템**

본 프로젝트는 **KPMG Future Academy 8기 3조(Iconic)**의 1차 프로젝트 기획안입니다. 중소 엔터테인먼트사의 고질적인 정산 지연 문제를 AI 기술로 해결하고자 합니다.
